# Student_Score_Prediction_ML
Predicting exam scores using Linear and Polynomial Regresiion with Python.
## Key Highlights
Baseline model used Linear Regression for initial score prediction
## Bonus Task 
Improved performance by adding features like Attendance and Sleep Hours using Polynomial Regression 
## Result:
Achieved an improved accuracy/R2 score of 54%

Libraries: Scikit_Learn, Pandas, NumPy, Matplotlib.
